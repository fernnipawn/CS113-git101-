Week  5 Functions and Program Structure
1. Scope Rules (ขอบเขตและช่วงชีวิตของตัวแปร) ในบางบริเวณของโปรแกรม ตัวแปรที่ถูกกำหนดขึ้นในบริเวณอาจถูกเข้าถึงได้ และเมื่อออกจาก บริเวณดังกล่าว ตัวแปรนั้นอาจไม่สามารถถูกเข้าถึงได้อีก
1.1 Local Variables
ตัวแปรที่ถูกประกาศไว้อยู่ภายใน block, ภายในตัวฟังก์ชั่น รวมถึงตัวแปรพารามิเตอร์ของฟังก์ชั่น จะเป็นตัวแปร local ซึงสามารถถูกใช้งานได้เพียงภายในฟังก์ชั่นที่ถูกประกาศไว้
1.2 Global Variables ตัวแปรที่ถูกประกาศไว้นอกฟังก์ชั่นเป็นตัวแปร global ซึ่งสามารถถูกเข้าถึงได้จากฟังก์ชั่นต่างๆ รวมถึงฟังก์ชั่น main ได้
1.3 Storage Classes   
 นอกจากการที่ตัวแปรมี ชื่อ ประเภท และค่าข้อมูล ได้แล้ว ภาษา C ยังกําหนดให้ผู้ ใช้สามารถกําหนด storage class ให้กับตัวแปรใน 4 ลักษณะ ผ่านการใช้คีย์เวิร์ด auto, register, extern, และ static เพื่อช่วยกำหนดช่วงชีวิตในการเก็บข้อมูลของตัวแปรนั้นใน memory ได้

2. Separate Compilation (การคอมไพล์แยกส่วน)
ส่วนของการนิยามฟังก์ชั่นและการเรียกใช้ไม่จำเป็นต้องถูกคอมไพล์ในเวลาเดียวกัน
วิธีการคอมไพล์ 
$ gcc -c main.c                   # คําสั่งนี้จะสร้าง main.o  
$ gcc -c func.c                   # คําสั่งนี้จะสร้าง func.o  
$ gcc main.o func.o -o main.exe   # คําสั่งนี้จะลิงก์ main.o และ func.o เข้าด้วยกัน


3. การใช้ make ไฟล์ Makefile จะบรรจุกฎต่างๆ ที่ใช้ในการสร้าง อาทิ object file หรือ executable file

6. Random Number Generator
•	rand() เป็นฟังก์ชั่นใน C Standard Library ที่ใช้ในการสร้างเลขสุ่มที่เป็นจำนวนเต็มที่อยู่ ระหว่าง 0 และ RAND_MAX ซึ่ง RAND_MAX เป็นค่าคงที่ที่ถูกกำหนดไว้ให้แล้วใน <stdlib.h>
•	ในโปรแกรม เวลามีการเรียกใช้ rand() หลายๆ ครั้ง ค่าที่ฟังก์ชั่นนี้ส่งกลับมาจะได้ตัวเลข จำนวนเต็มที่แตกต่างกันออกไปเป็นการสุ่มนั่นเอง
