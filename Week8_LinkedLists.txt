Week  8 Linked Lists
1. Linked List   คือLinked list เป็นโครงสร้างข้อมูลที่ถูกออกแบบมาใช้เก็บข้อมูล โดยที่ข้อมูลแต่ละตัวจะเชื่อมโยงต่อกันเป็น สาย เพื่อผู้ใช้สามารถเข้าถึงข้อมูลแต่ละตัวได้  Array ก็เป็นโครงสร้างข้อมูลที่มีลักษณะที่ข้อมูลแต่ละตัวอยู่เรียงต่อกัน แต่ทั้ง array และ linked list มีโครงสร้างและการจัดการในการเก็บข้อมูลทีไม่เหมือนกันหมด

2.  Structure Structure  เป็นการกำหนดรูปแบบหรือโครงสร้างข้อมูลทีมีการรวบรวมตัวแปรที่เก็บข้อมูลหลายเรื่องที่ สัมพันธ์กัน ที่อาจมีประเภทข้อมูลที่แตกต่างเข้าไว้ด้วยกัน ตัวอย่างการกำหนด structure
struct student {    
    char id[12];    
    char name[40];    
int age; }

2.1 การประกาศตัวแปรประเภท Structure
เราสามารถประกาศตัวแปรที่มีโครงสร้าง structure ได้ดังนี้
ตัวอย่าง    struct student another_std = {"6211140123", "Josh Smith", 18};

2.2 การใช้ typedef เราสามารถกำหนดประเภทข้อมูลขึ้นมาเพิ่มเติมผ่านการใช้คีย์เวิร์ด typedef ได้ ตัวอย่างการใช้งาน   typedef unsigned long int ulint;
ในส่วนของ Linked Lists นี้ เราสามารถเพิ่มหรือลดข้อมูลลงไปในนี้ได้ ในการเพิ่มข้อมูลนั้นก็มีทั้งการเพิ่มไปที่ส่วนต้น และส่วนกลางของlist
นอกจาก Singly Linked List เรายังสามารถสร้างโครงสร้างข้อมูลทีเรียกว่า Doubly Linked List โดยที่แต่ละโหนดที่อยู่ติดกันเชื่อมต่อกันด้วยลิงก์ 2 ลิงก์ ลิงก์อันนึงใช้เชื่อมต่อไปยังโหนดถัดไป และลิงก์อีกอันนึงใช้เชื่อมต่อไปยังโหนดก่อนหน้า
ส่วน Circular Linked List ก็จะวนจากส่วนhead ไปจนสุดแล้วก็จะวนกลับมาที่ส่วนheadเหมือนเดิมการวนจะวนเป็นวงกลม
